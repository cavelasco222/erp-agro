ERP Agro

Para comenzar necesitás configurar dos cosas: el Google Client ID para el login y la URL del Apps Script para guardar datos.

  1. Apps Script: abrí el Google Sheet → Extensiones → Apps Script → pegá el código → publicá como Web App → copiá la URL
  2. Google Client ID: entrá a console.cloud.google.com → APIs → Credenciales → OAuth 2.0 → creá uno para Web → copiá el Client ID
  3. Pegá ambos abajo y hacé click en Conectar
  4. O hacé click en Modo demo para explorar sin guardar datos

Los datos se leen desde tu Google Sheet. Para escritura se necesita el Apps Script.
